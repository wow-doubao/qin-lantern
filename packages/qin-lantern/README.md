# Qin Lantern

> 秦灯笼产自中国的陕西，形状像灯笼，有着独特的纹路和刺状突起，是一种独特的文玩收藏品。

啊，不。

这是一套基于Element Plus封装的组件库...

### 食用方式

With NPM:

```bash
$ npm install qin-lantern
```

With Yarn:

```bash
$ yarn add qin-lantern
```

With PNPM:

```bash
$ pnpm install qin-lantern
```
