qlLocale 属性用于全局国际化配置，组件自带 中（默认）、英 两个语言包，也可扩展其他语言包。

import { zhCn, en } from 'qin-lantern/locale'

国际化配置示例： ​

ts
export default {
  name: 'zh-cn',
  ql: {
    dialog: {
      confirm: '确认',
      cancel: '取消',
      loading: '请求中'
    },
    paging: {
      loading: '加载中',
      clickMore: '点击加载更多'
    },
    upload: {
      fileSize: '文件大小不能超过',
      uploadError: '上传失败，请稍后重试'
    },
    formList: {
      input: '请输入',
      select: '请选择',
      more: '更多信息'
    }
  }
}
